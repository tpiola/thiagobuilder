# Contribuindo

## Fluxo

- `main`: produção
- `develop`: integração
- `feat/*` e `fix/*`: branches curtas por mudança

## Padrão de commits

- `feat: ...`
- `fix: ...`
- `chore: ...`
- `docs: ...`
- `test: ...`

## Antes de abrir PR

```bash
pnpm check
pnpm lint
pnpm test
pnpm build
pnpm e2e
pnpm a11y
```
